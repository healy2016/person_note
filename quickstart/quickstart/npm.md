### 查看全局所有的依赖，一级目录下
> * npm list -g --depth 0

### 开发环境下的依赖安装
> * npm install XXX --save-dev

### 生产环境下的依赖安装
> * npm install XXX --save

### 使用淘宝镜像的安装方法
> * npm --registry https://registry.npm.taobao.org install express

### 安装cnpm直接使用淘宝镜像：必须是全局安装-g
> * npm --registry https://registry.npm.taobao.org install -g cnpm
